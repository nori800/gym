# 用語集

| 用語 | 説明 |
|------|------|
| MVP | 最小限の機能で価値検証できる最初のリリース範囲 |
| PWA | Progressive Web App。ブラウザからホーム画面追加などネイティブに近い体験を目指すWebアプリ |
| RLS | Row Level Security。Postgres で行単位のアクセス制御を行う仕組み（Supabase で認可の中核） |
| Supabase Auth | メール／OAuth 等の認証。JWT を通じて DB・Storage と連携 |
| Route Handler | Next.js App Router の `app/api/.../route.ts` におけるサーバー側 HTTP ハンドラ |
| 補助線 | 動画上映像の上に重ねるガイドライン（グリッド・中心線など） |
| RPE | Rate of Perceived Exertion。主観的な負荷の指標としてトレーニング記録に用いる想定 |
| getUserMedia | ブラウザからカメラ／マイクへアクセスする API。撮影機能で利用 |
| MediaRecorder | ブラウザでメディアストリームを録画・録音する API。実装可否はプラットフォームによる |
