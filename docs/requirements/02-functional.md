## 6. 主要ユースケース

### 6-1. 動画確認
- ユーザーは**アプリ内カメラで動画を撮影**できる（ステップ1）
- ユーザーは撮影済み／ライブラリの動画をアップロードする
- ユーザーは動画を再生・一時停止する
- ユーザーは撮影中・プレビュー時・再生時に補助線（16分割を含む）をONにして姿勢を確認する
- ユーザーは任意のフレームで止めて確認する
- ユーザーは気づきをメモする

### 6-2. 記録管理
- ユーザーは種目、重量、回数、セット数を入力する
- ユーザーは動画とトレーニング記録を紐付ける
- ユーザーは過去履歴を見返す

### 6-3. 比較（Phase2）
- ユーザーは前回動画と今回動画を並べて比較する
- ユーザーはフォーム改善を確認する

## 8. 機能要件

### 8-1. 認証機能
- メールアドレス＋パスワードで登録・ログインできる
- Googleログインに対応できる設計にする
- 未ログイン時の扱いはポリシーに従う（全機能ロック／撮影のみローカル保存など）。クラウド同期・共有データが必要な操作ではログインまたは登録へ誘導する
- ログアウトできる
- パスワード再設定できる
- 画面ステップ上はログイン・新規登録・初期設定を **次のステップ群** とし、[08-user-flow-and-steps.md](08-user-flow-and-steps.md) と整合させる

### 8-2. ユーザープロフィール
- 表示名を設定できる
- 身長、体重、目的を登録できる
- 利き手を設定できる
- よく使う種目を設定できる

### 8-3. 動画の取得・アップロード
- **アプリ内カメラで動画を撮影**できる（`getUserMedia` / MediaRecorder 等。HTTPS 必須）
- 撮影中および撮影直後のプレビュー画面で、**16分割グリッドの表示 ON/OFF** を常に操作できる。**初期状態は 16分割表示 ON**（ユーザーがOFFに切り替え可能）
- 端末ライブラリから動画を選択できる
- 動画をSupabase Storageへ保存できる（認証ポリシーに合わせたタイミング）
- 動画にタイトルをつけられる
- 種目タグを設定できる
- 撮影日を設定できる
- メモをつけられる
- 動画を削除できる

### 8-4. 動画一覧
- 自分の動画一覧を表示できる
- サムネイルを表示できる
- 種目で絞り込める
- 撮影日順に並べ替えできる
- お気に入り絞り込みできる（Phase2）

### 8-5. フォーム確認
- 動画を再生できる
- 一時停止できる
- 5秒送り / 5秒戻しができる
- フレーム送り / 戻しができる
- 再生速度変更（0.25 / 0.5 / 1.0）ができる
- 全画面表示できる

### 8-6. 補助線機能
- 16分割グリッドを表示できる（**撮影・プレビュー・再生**の各局面で要件に応じて利用。**撮影時は 8-3 の ON/OFF・初期 ON を満たす**）
- 中心縦線を表示できる
- 中心横線を表示できる
- 補助線のON/OFFができる
- 補助線の色を変更できる
- 補助線の太さを変更できる
- 補助線の透明度を変更できる

### 8-7. 描画機能
- 動画上に直線を引ける
- 動画上に矢印を引ける
- 動画上に丸を描ける
- 描画を削除できる
- 描画内容を保存できる
- 描画入り状態でスクリーンショット保存できる

### 8-8. メモ機能
- 動画単位でメモを保存できる
- 任意フレームに対する気づきを保存できる設計にする
- メモを編集・削除できる

### 8-9. トレーニング記録
- 日付を登録できる
- 種目名を登録できる
- 重量を登録できる
- 回数を登録できる
- セット数を登録できる
- RPEを登録できる
- コメントを登録できる
- 記録に動画を紐付けられる

### 8-10. 履歴閲覧
- 種目別に履歴を見られる
- 日付順で履歴を見られる
- 動画つき履歴のみ見られる
- 前回記録を表示できる
- 自分のメモを振り返られる

### 8-11. 比較機能（Phase2）
- 2本の動画を選択できる
- 並列表示できる
- 同時再生できる
- 片方だけ一時停止できる
- 両方に補助線表示できる
- Before / After 比較ができる

### 8-12. 共有機能（Phase2）
- 動画または記録の共有リンクを発行できる
- 公開範囲を限定できる
- 将来的にトレーナーコメントを受け取れる設計にする
